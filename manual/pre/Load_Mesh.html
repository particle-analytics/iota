Loading a Mesh

Loading a Mesh into Pre-Processing

Users can load meshes into the pre-processing module by clicking on Load Mesh button (see Figure 27). This button gives access to the list of meshes that are associated with the current scenariro and/or the current project. In order to load a mesh from the list, users just need to click on the mesh to be loaded. Furthermore, users can also load meshes by using the option "Import Mesh from File".
