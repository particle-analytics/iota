Files

Files

This section shows the files and scripts associated with the project/scenario. The "Files" accordion shows the following information per file (see Figure 13):

  • Name: name provided by the user to indentify the mesh.
  • Type: Files are grouped into two different types:
    • Scripts: python script files (.py).
    • Other: any other file to be associated with the project.
  • Filename: name of the file on the hard drive.
  • Scenario: name of the scenario if the file is associated with a scenario.
  • Modfied: latest modification date of the file/script.

Figure 13 - "Files" section of Iota

Importing a Script/File

Users can import files or scripts into the project/scenario by clicking on the "Import" button located at the right corner of accordion (see Figure 13). A pop-up window will be shown in order introduce the following fields for the file/script:

  • Name: users needs to provide a name for the file/script.
  • Description (optional): users can provide a description for the file/script.
  • Scenario (optional): users can associate the file/script to an existent scenario of the project.
  • Type: users can select between two type of files:
    • Scripts: files with extension .py.
    • Other: any other kind of file that it is not a script.
  • File: the icon located at the right side opens a file browser window that allows users to select the file/script to be imported.
