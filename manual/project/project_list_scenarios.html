List of Scenarios

List of Scenarios

This section shows the list of the scenarios contained in the project. The following basic information of each scenario is shonw in the list:

  • Name: name assigned to the scenario.
  • Description: brief description of the scenario.
  • Created: date of creation of the scenario.
  • Modified: latest date when the scenario was modified.

Figure 17 - "Scenarios" section at the "Project" page

Loading a Scenario

Users can load a scenario by means of the Load button that shows up when users are hovering over the list of scenarios (see Figure 17). After clicking on the Load button, the scenario will be autotmatically loaded and the "Scenario" page will be shown on the screen.

Creating a New Scenario

Users can create new scenarios by using the button Create located at the top righ corner of the list of the scenarios at the project page (see Figure 17). After clicking on the button Create, a pop-up window is shown in order to introduce the following information for the new scenario:

  • Name: users need to provide a name for the scenario.
  • Description (optional): users can provide a brief description for the scenario.

Figure 18 - Create Scenario window

Cloning a Scenario

Users can clone a scenario in order to create a new scenario with the same settings that the cloned one:

  • Click on the scenario to be cloned from the list of scenarios.
  • Click on the button Clone located at the top righ corner of the list of the scenarios.

  • The "Clone Scenario" window shows up in order to introduce the following information for the new scenario:

    • Name: user need to provide a name for the scenario (by default the name of the scenario to be cloned is shown).
    • Description (optional): users can provide a brief description for the scenario (by default the description of the scenario to be cloned is shown).

Figure 19 - Clone Scenario window
