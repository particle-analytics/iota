Post-processing

Post-processing

The post-processing module of Iota allows the users to visualise the results contained in the datasets and investigate the data by means of several post-processing tools. The following sections can be indentified in the post-processing module (see Figure 58:

  1. Rendering List: This section shows the list of items (meshes and/or particles) that are currently loaded in the post-processing module.
  2. Tools menu:
    • Results: allows users to select the results to be visualised.
    • Tools: allows users to perform post-processing operations with the results.
      • Cut Plane
      • Line Variation
      • Point Evolution
      • Integrals
      • Statistics
      • Stream/Path
  3. Rendering area
  4. Views toolbar
  5. Animation toolbar
  6. Bottom toolbar:
    • Screenshot: it allows users to automatically create an screenshot of the rendering area.
