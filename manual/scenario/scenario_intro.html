Scenario

Scenario

The "Scenario" page provides information about the assets associated with the current scenario. The structure of the page is similar to the "Project" page (see Figure 20):

  1. Title and Description of the scenario.
  2. Global: this section shows rectangular accordions for the different type of assets associated with the scenario:
    • Datasets
    • Screenshots
    • Graphs
    • Videos
    • Files
  3. Variables: this section shows the list of project's variables and their values for the current scenario.
  4. Thumbnail: users can select an image to be used as thumbnail for the scenario.
  5. Additional information:
    • User Name: name of the user who created the project.
    • Created Date: project's creation date.
    • Last Modified: Latest data when the project was modified.
    • Tags

Figure 20 - Overview of the "Scenario" page

Editing the Info of a Scenario

The basic info of the scenario ("Title", "Description" and "Tags") can be edited by using the option "Edit Scenario". Users can access to this options clicking on the "More" button located at the rigth side of the page (see section 5 in Figure 20).

Deleting a Scenario

Users can delete an existing scenario from their database by means of the option "Delete Scenario". This option is accessible after clicking on the "More" button located at the right page (see section 5 in Figure 20)

Updating the variables' values for the current scenario

Users can update the value of the variables shown in the scenario by clicking on the button Edit (for "Manual" variables) or on the button Calculate (for "User-Defined" and "Bulit-in" in variables). These buttons are shown when users are hovering around the listed variables.
