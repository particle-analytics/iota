Dashboard

Dashboard

The "Dashboard" page of Iota provides an overview of the list of projects contained in the database of the user. Each project is represented as a rectangular accordion that shows the title of the project and the last date when the project was modified (see Figure 1).

Users can filter the list of projects by means of the "Search by title" bar located at the top of the windonw. Moreover, the projects can be sorted by different options (Title, Created Date, Modified Date and Last Accessed Date) using the combo box located at the right side of the search bar.

Users can get more detailed information about the contents of the projects by click on their correspondent accordion:

  • Description
  • Created date
  • Number of items of each project's assets type:
    • Scenarios
    • Screenshots
    • Videos
    • Graphs
    • Meshes
    • Variables
  • Tags

Figure 1 - "Dashboard" page where the list of projects is shown

Creating a new project

A new project can be created by clicking on Add Project button located at top left of screen (see Figure 1). After clicking on the button, the "Create Project" window shows up with the following fields:

  • Name: users need to provide a name for the project.
  • Description (optional): users can provide a description for the project.
  • Tags (optional): mulitple tags/keywords can be associated with the project by using commas (for instace: DEM, COHESIVE, SILO)

Figure 2 - Pop-up window to create a new project

Editing a project's info

The basic info of the project ("Title", "Description" and "Tags") can be edited by using the option "Edit Project". This is accessible by clicking on the "More" button located at the right corner of the projects' accordions.

Deleting a project

Users can delete an existing project from their database by means of the option "Delete Project" that is accessible after clicking on the "More" button located at the right corner of the projects' accordions.

Loading a project

Users can load projects by means of the "Load" button that appears at the right corner of the projects' accordions (see Figure 1). After clicking on the "Load" button, the project will be autotmatically loaded and the "Project" page will be shown on the screen.
